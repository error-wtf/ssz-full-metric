# Contributing to SSZ Full Metric

Thank you for your interest in contributing to this revolutionary black hole solution!

## 🌟 Ways to Contribute

- **Bug Reports** - Found an issue? Open a GitHub issue
- **Feature Requests** - Ideas for improvements? We'd love to hear them
- **Code Contributions** - Pull requests welcome!
- **Documentation** - Help improve our docs
- **Testing** - Run tests on different platforms
- **Research** - Extend the theory, validate predictions

## 📝 Guidelines

1. **Code Style** - Follow PEP 8 for Python
2. **Tests** - All new features need tests
3. **Documentation** - Update docs for new features
4. **Commits** - Clear, descriptive commit messages
