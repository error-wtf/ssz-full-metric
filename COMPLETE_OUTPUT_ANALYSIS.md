### **Issue 2: Ξ(r) vs A(r) Inconsistency**

```
From Ξ(r) = 1 - exp(-φ·r/r_s):
D(r) = 1/(1 + Ξ)
A(r) = D(r)² (from GR: D = √A)
→ A(r) = 1/(1 + Ξ)²

From Post-Newtonian:
A(r) = 1 - 2U + 2U² - 4.8U³

In weak field (U << 1):
Ξ ≈ φ·r/r_s = 2φU ≈ 3.24U
D ≈ 1/(1 + 3.24U) ≈ (1 - 3.24U)
A ≈ (1 - 3.24U)² ≈ 1 - 6.48U

But we want: A ≈ 1 - 2U

→ MISMATCH by factor ~3! ⚠️⚠️⚠️

Resolution Options:
A) Different normalizations (Ξ_max ≠ 1?)
B) Extra factor in Ξ(r) formula
C) A(r) ≠ D(r)² in SSZ
D) Ξ(r) describes something else

MUST RESOLVE! ⚠️⚠️⚠️
```
